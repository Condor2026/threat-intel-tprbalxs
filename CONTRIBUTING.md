# Guía de Contribución

## 🛡️ Threat Intel - tprbalxs.shop Campaign

Gracias por tu interés en contribuir a este proyecto de inteligencia de amenazas.

### 📋 Cómo Contribuir

1. **Reportar nuevos IOCs**
   - Si detectas nuevos dominios asociados a esta infraestructura, añádelos a la lista en `/iocs/domains.txt`
   - Incluye la fecha de detección y la fuente

2. **Mejorar reglas de detección**
   - Las reglas YARA, Suricata y Snort están en `/signatures/`
   - Si encuentras falsos positivos o mejoras, abre un Issue o Pull Request

3. **Documentación**
   - Si encuentras errores o quieres añadir más contexto, edita el README

4. **Scripts de detección**
   - Los scripts están en `/detection/`
   - Asegúrate de que funcionan en múltiples plataformas

### 🔒 Política de Seguridad

- **NO** publiques información personal de víctimas
- **NO** publiques exploits o código malicioso funcional
- **NO** realices ataques activos contra la infraestructura reportada

### 🧪 Requisitos para Pull Requests

- Describe claramente el cambio
- Añade ejemplos o pruebas si es posible
- Sigue el formato de los archivos existentes

### 📧 Contacto

Para reportes de seguridad o consultas, abre un Issue en GitHub.
