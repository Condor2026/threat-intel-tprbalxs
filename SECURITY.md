# 🔒 Política de Seguridad

## Reporte de Vulnerabilidades

Si encuentras una vulnerabilidad en este repositorio o en la investigación, por favor:

1. **NO** abras un Issue público.
2. Envía un correo a: **KiraSecuruty@proton.me** 
3. Espera confirmación antes de divulgar públicamente.

## IOCs Desactualizados

Los Indicadores de Compromiso (IOCs) pueden cambiar rápidamente. Si detectas que un IOC ya no es válido, abre un Issue o Pull Request para actualizarlo.

## Reporte de Nuevos IOCs

Si detectas nuevos dominios, IPs o hashes relacionados con esta campaña, por favor:

1. Añádelos a los archivos correspondientes en `/iocs/`
2. Incluye la fecha de detección
3. Aporta evidencia (capturas, logs, etc.)

## Código de Conducta

Este proyecto sigue el [Código de Conducta de GitHub](https://docs.github.com/es/site-policy/github-terms/github-community-code-of-conduct).
